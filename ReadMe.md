****A Vioce Assistance Using AWS Api****

**The structure of the repository**

The five folders respectively answers the five requests of the race. The 'talker.py' in each of the folders integrates the other python files to create a real usable talker APP, but the AWS api-key may not be usable now, so you cannot run the main code now. The RAG uses Faiss and BM25 to calculate the score of relativeness of the paragraphs of 'The Story of A Stone'. The OCR uses a GOT package available on github to achieve CV --- Able to recognize characters in pictures. The other requests' realization are easier, such that I won't describe them from here.
